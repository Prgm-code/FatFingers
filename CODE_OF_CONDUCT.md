# Code of Conduct

Este proyecto busca ser un espacio tecnico respetuoso y productivo.

## Comportamiento esperado

- Ser claro y respetuoso.
- Discutir ideas y codigo, no personas.
- Dar feedback accionable.
- Aceptar correcciones cuando haya nueva informacion.
- Cuidar la privacidad de usuarios y contribuidores.

## Comportamiento no aceptado

- Acoso, insultos o ataques personales.
- Comentarios discriminatorios.
- Publicar informacion privada de otra persona.
- Publicar secrets, tokens o datos sensibles.
- Spam o discusiones fuera de tema repetidas.

## Aplicacion

Los maintainers pueden editar, ocultar o cerrar comentarios, issues o PRs que violen este documento. Tambien pueden bloquear contribuciones si el comportamiento compromete la seguridad o la colaboracion del proyecto.
